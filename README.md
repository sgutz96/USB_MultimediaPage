# Ingeniería Multimedia — USBBOG

## Estructura del proyecto
```
multimedia_web/
├── index.html
├── css/
│   ├── style.css        ← tokens, tipografía, botones, grid blueprint
│   ├── navFooter.css    ← nav + footer
│   └── home.css         ← todas las secciones
├── js/
│   ├── about.js         ← particles.js + scroll reveal
│   └── aboutThree.js    ← Three.js terrain 3D
└── assets/
    └── img/
        ├── Logo.png
        ├── Img_1.png
        └── Banner/
            ├── Banner_ (1).jpg
            ├── Banner_ (2).jpg
            └── Banner_ (3).jpg
```

## Instrucciones
1. Copiar tus imágenes en `assets/img/`
2. Abrir `index.html` en un servidor local (Live Server en VS Code, por ejemplo)
3. No abrir como archivo directo — Three.js requiere servidor HTTP por los módulos ES
